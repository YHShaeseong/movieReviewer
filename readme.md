# MovieFinder

취향에 맞는 영화를 추천받는 웹사이트

## 주요 기능

### 1. 영화 검색
- 상단 검색창에서 영화 제목 입력
- 실시간으로 결과 표시

### 2. 장르 필터
- 액션, 드라마, 코미디 등 8개 장르 선택
- 클릭하면 해당 장르 영화만 표시

### 3. TOP100 영화 목록
- 평점 높은 영화 순으로 표시
- 각 영화의 제목, 연도, 평점, 좋아요 수 확인 가능

### 4. 메인 배너
- 인기 영화 소개
- 예고편 및 상세정보 버튼

## 파일 구조

- `index.html` - 웹페이지 구조
- `style.css` - 디자인 스타일
- `script.js` - 검색/필터 기능

## 실행 방법

`index.html` 파일을 브라우저에서 열기
